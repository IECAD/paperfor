# 논문 #51: Recognition of Blocking Categories for UWB Positioning in Complex Indoor Environment (Sensors, 2020)

## 1) 기본 정보
- **제목:** Recognition of Blocking Categories for UWB Positioning in Complex Indoor Environment
- **저자:** Yaguang Kong, Chuang Li, Zhangping Chen, Xiaodong Zhao
- **저널/학회:** Sensors
- **출판연도:** 2020
- **DOI/링크:** 10.3390/s20154178
- **인용 횟수:** —

**3줄 요약**
1. **CIR 특징 선택**을 위해 **최대 상관·최소 중복·최소 계산비용(BS-mRMRMC)**를 제안.  
2. **DT-SVM 계층 분류기**로 LOS/NLOS뿐 아니라 **차폐 종류(물/인체/금속/벽/목재)**까지 인식.  
3. 카테고리 평균 **정확도 96.7%**로 기존 방법 대비 향상.

**핵심 기여(Contribution)**

- [x] **차폐 카테고리 인식**으로 공간 인지력 강화
- [x] **DT-SVM** 계층 구조로 높은 정확도

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - 데이터 타입: UWB **CIR**, TDoA 기반 측정(DW1000)
  - 차원: 상태/카테고리 분류
  - 샘플링: Train 60k / Test 6k 샘플(6개 클래스)
- **전처리 (pseudo)**
  1) CIR 30 포인트 추출 → 2) 특징 후보 계산 → 3) **BS-mRMRMC**로 최적 특징 집합 선택

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| CIR 통계 | 최적 특징 집합 | 공통 set 교집합 추출 |
| 품질확률 | DT-SVM 출력 | R/Q 스케일러로 매핑 |
| NLOS 유형 | 차폐 카테고리 | 가중/게이팅 규칙화 |

- **호환성 평가**: [x] 직접 사용 가능
