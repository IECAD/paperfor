# 논문 #28: *UWB Sensor-Based Indoor LOS/NLOS Localization With Support Vector Machine Learning* (IEEE Sensors Journal, 2023-02-01)

## 1) 기본 정보
- **제목:** UWB Sensor-Based Indoor LOS/NLOS Localization With Support Vector Machine Learning
- **저자/소속:** Hongchao Yang et al.
- **저널/학회/연도:** IEEE Sensors Journal, 2023
- **링크/DOI:** —
- **키워드:** UWB, CIR, CCPs, DTC, Fuzzy Credibility, **SVM**, NLOS 보정

**3줄 요약**
1) **새로운 CIR 특성 3종(CCPs)**: *False Crests Number(FCN), FP Error(FPE), FP Distance Error(FDE)*를 도입.  
2) **2단계 식별**: **DTC(동적 임계)**로 1차 분류 → **FC-SVM**으로 미세 분류.  
3) 분류 결과로 **실시간 거리오차 보정**을 설계해 **평균 위치오차 0.250→0.091 m(63.49%↓)**, LOS/NLOS 식별 **평균 93.27%**, LOS/NLOS 재현율 **94.27/92.57%**.

**핵심 기여(Contribution)**
- [x] FP 탐지과정을 **ENS/FP/MP 3단계**로 모델링하고 이에 맞춘 **신규 CCPs** 제안  
- [x] **DTC→FC-SVM** 2단계 분류로 **환경 변화**에 견고  
- [x] 분류 결과 기반 **범위오차 보정**으로 **LOS/NLOS 모두** 성능 향상

---

## 2) 입력 데이터 (Input)
- **원본 입력:** UWB **DS-TWR** 거리 + **CIR 파형**  
- **전처리(요약):** LDE 임계 \(L=S\times NTM\) 기반 FP/MP 분할 → CCPs 추출(시간·에너지)

**우리 20D 입력과의 매핑**

| 우리 20D 특징 | 논문 특징 | 변환/활용 |
|---|---|---|
| CIR/품질채널 | CCPs(FCN/FPE/FDE 등) | **분류 확률→Rₜ 스케일**·잔차게이팅 |
| Range | 분류 기반 보정 | **보정함수/회귀**와 하이브리드 |
| 맵/IMU | — | 맵가시성/IMU 품질과 **메타-특징** 융합 |

- **호환성 평가:** ☑ **직접 사용 가능**

---

## 3) 모델 아키텍처 (Architecture)
- **파이프라인:** [CIR→CCPs] → **DTC**(coarse) → **FC-SVM**(fine) → **NLOS 거리 보정** → 위치  
- **핵심 수치:** 범위오차(LOS **0.106→0.065 m**, NLOS **1.442→0.739 m**)

**우리 모델과 비교**

| 구성 요소 | 논문 | 우리 | 차이/시너지 |
|---|---|---|---|
| NLOS 판별 | DTC+FC-SVM | 퍼지/DPGMM/LSTM | **앙상블 스태킹** |
| 보정 | 규칙형 보정 | 회귀·맵가중 | **하이브리드 보정** |
| 백엔드 | LS/WLS | EKF/UKF/PF/FG | **프런트엔드 모듈화** |

## 4) 학습/설정
- **분류기:** SVM(FC-SVM), DTC 임계는 **식별 결과로 갱신**
- **라벨:** LOS/NLOS
