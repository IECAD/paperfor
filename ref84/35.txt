# 논문 #35: *A Machine Learning Approach to Improve Ranging Accuracy with AoA and RSSI* (Sensors, 2022)

## 1) 기본 정보
- **제목:** A Machine Learning Approach to Improve Ranging Accuracy with AoA and RSSI
- **저자/소속:** —
- **저널/학회/연도:** Sensors, 2022
- **링크/DOI:** —
- **키워드:** **Wi-Fi AoA+RSSI**, **ANN** 거리추정, 멀티패스 완화, 저복잡

**3줄 요약**
1) **AoA 추정 + RSSI**를 **ANN**에 입력해 **LoS 거리**를 회귀 → **멀티패스 오차**를 현저히 감소.  
2) **넓은 대역·엄격 동기** 없이도 **< 10 cm** 수준의 **거리 오차** 달성.  
3) 단일 AP 기반 구현 가능, **저비용/저복잡**을 유지.

**핵심 기여(Contribution)**
- [x] **AoA+RSSI 결합 ANN**으로 **고정밀 거리**  
- [x] **멀티패스 보정**(RSSI 단독 한계 극복)  
- [x] 간단한 인프라(단일 AP)로 성능 확보

## 2) 입력 데이터 (Input)
- **원본 입력:** **AoA 스펙트럼**, **RSSI**  
- **전처리:** ANN 회귀용 정규화/특징화

**우리 20D 입력과의 매핑**

| 우리 20D 특징 | 논문 특징 | 변환/활용 |
|---|---|---|
| Range | ANN-보정 거리 | **UWB 대체/보조** 거리로 하이브리드 |
| 품질 | 멀티패스 민감 | **품질지표**로 가중/게이팅 강화 |
| 멀티센서 | Wi-Fi | UWB 부족/불가 지역 **보완** |

- **호환성 평가:** ☑ **직접 사용 가능**(하이브리드 거리 소스)
