# 논문 #33: *A CSI-Based Indoor Positioning System Using Single UWB Ranging Correction* (Sensors, 2021)

## 1) 기본 정보
- **제목:** A CSI-Based Indoor Positioning System Using Single UWB Ranging Correction
- **저자/소속:** —
- **저널/학회/연도:** Sensors, 2021
- **링크/DOI:** —
- **키워드:** **Wi-Fi CSI 지문 + CNN**, **단일 UWB 앵커** 보정, **SVR+IMU** 보정, PDR 초기화

**3줄 요약**
1) **초기 단계:** IMU-PDR로 경로 생성·지문 수집 → **CNN 회귀 모델** 학습. 동시에 **IMU 증분↔단일 UWB 거리 증분** 상관으로 **SVR 사전학습**.  
2) **운영 단계:** CNN 위치를 **보정된 UWB ranging**으로 **주기적 재보정** → **장기 안정성** 확보.  
3) **장점:** **저비용**, **ML 비사용(핵심 모듈 최소화)** 현장 적용성↑, **NLoS 적응 보정**.

**핵심 기여(Contribution)**
- [x] **UWB 한 개**만으로 **CNN 위치 재보정 루프**  
- [x] **SVR+IMU**로 **NLoS 영향**을 보정한 **거리 회복**  
- [x] **장기 안정 운영**(환경 변화에 점진 적응)

---

## 2) 입력 데이터 (Input)
- **Wi-Fi CSI**(지문) + **IMU**(PDR) + **UWB 단일 거리**
- **전처리:** PDR 기반 **라벨·지문** 수집, UWB-IMU **증분 상관**으로 SVR 학습

**우리 20D 입력과의 매핑**

| 우리 20D 특징 | 논문 특징 | 변환/활용 |
|---|---|---|
| Fingerprint | CSI→CNN | **지문 채널** 확장(우리 맵/품질과 병행) |
| Range | 단일 UWB | **보정 루프**로 위치 drift 억제 |
| IMU | PDR | 초기화/보정 동시 활용 |

- **호환성 평가:** ☑ **직접 사용 가능**(보정 루프 모듈)

---

## 3) 모델 아키텍처 (Architecture)
- **2-Stage:** **[초기] PDR·지문수집/CNN·SVR 학습 → [운영] CNN 예측 + UWB-보정**  
- **핵심:** **주기적 역전파 보정**으로 **장기 성능 유지**

**우리 모델과 비교**

| 구성 요소 | 논문 | 우리 | 시너지 |
|---|---|---|---|
| 지문 | CSI+CNN | 지도/맵/품질 | **보정 루프**만 도입 |
| 보정 | UWB 1개 | 다앵커 UWB | **혼합 운영** 가능 |
| IMU | PDR | INS/PDR | 공통화 쉬움 |
