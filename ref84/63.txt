# 논문 #63: NLOS identification using parallel deep learning model and time-frequency information in UWB-based positioning system (Measurement, 2022)

## 1) 기본 정보
- **제목:** NLOS identification using parallel deep learning model and time-frequency information in UWB-based positioning system
- **저자:** Junyu Wei, Haowen Wang, Shaojing Su, Ying Tang, Xiaojun Guo, Xiaoyong Sun
- **저널/학회:** Measurement 195
- **출판연도:** 2022
- **DOI/링크:** 10.1016/j.measurement.2022.111191
- **인용 횟수:** —

**3줄 요약**
1. **MIL(멀티 입력 학습)**: **원시 CIR(1D)** + **TFDOCIR(2D, CWT-기반 시간-주파수 이미지)** **병렬** 특징 추출(CIR→**GRU×3**, TFDOCIR→**CNN×4**).  
2. **장애물 5종(목문/콘크리트/금속/인체/유리)**에서 **개별 정확도 86.82–92.91%**, **전체 91.74%**.  
3. **WLS**로 위치결정 연결 → **평균 위치오차 7.35 cm**.

**핵심 기여(Contribution)**
- [x] **CIR+TFDOCIR** 동시 활용(MIL)로 정보 보강
- [x] **CWT(일반화 Morse)**로 TFDOCIR 구성
- [x] **WLS** 연계로 **실위치** 성능 검증

---

## 2) 입력 데이터 (Input)
- **원본 입력:** 원시 **CIR**, **TFDOCIR**(CWT 스펙트로그램)
- **전처리 (pseudo)**
  1. CIR 정규화 → **GRU×3**  
  2. CIR→**CWT**(scale 128 등) → **TFDOCIR** → **CNN(Conv+Pool×4, Flatten)**  
  3. [CIRfeat ‖ TFDfeat] → Hidden → **Sigmoid**(이진: LOS/NLOS)

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 |
|---|---|---|
| CIR/파생 | GRU·CWT·CNN 병렬 | 동일 적용 |
| 품질/가중 | p(LOS) | **R 게이팅/WLS 가중** |
| 보정 | — | 보정 모듈 전단 게이트

- **호환성 평가:** [x] 직접 사용 가능

---

## 3) 모델 아키텍처 (Architecture)
[CIR→**GRU×3**] ‖ [**TFDOCIR→CNN×4**] → [Merge] → [Hidden] → [Sigmoid]

**우리 모델과 비교**

| 구성 | 논문 | 우리 | 차이 |
|---|---|---|---|
| 병렬 입력 | CIR+TFDOCIR | CIR/통계/WT 등 | 동일/확장 |
| 백엔드 | GRU‖CNN | CNN/TCN/LSTM | 호환 |

---

## 4) 학습 방법 (Training)
- **손실:** CE  
- **세부:** k=LR/옵티마 표준(논문 설정) / 파라미터 업데이트 규정식 명시

## 5) 실험 및 결과 (Experiments)

|---|---:|

| 장애물별 Acc | **86.82–92.91%** |
| WLS 평균 위치오차 | **7.35 cm** |
