# 논문 #25: *Accurate Indoor Positioning for UWB-Based Personal Devices Using Deep Learning* (IEEE Access, 2023)

## 1) 기본 정보
- **제목:** Accurate Indoor Positioning for UWB-Based Personal Devices Using Deep Learning
- **저자/소속:** —
- **저널/학회/연도:** IEEE Access, 2023
- **링크/DOI:** —
- **키워드:** UWB, TWR, CIR, Deep Learning, NLOS 확률, 가중 Kalman

**3줄 요약**
1) **CIR→DL**로 프레임별 **NLOS 확률**을 추정해 **Kalman 측정공분산 R/이득 K**를 동적으로 조정.  
2) **±25 cm / ±50 cm** 허용범위 정확도 **20.84–27.22% / 7.78–22.78%** 개선(기본 KF/삼변측량 대비).  
3) 포켓/손차폐 등 **동적 사용자 자세**에서 추적 연속성과 안정성이 향상.

**핵심 기여(Contribution)**
- [x] CIR 기반 **DL 신뢰도 추정**으로 **가중 KF** 실시간 적용
- [x] **간단한 통계특징(에너지/MA/Trise/τm/왜도/첨도)**로 경량 구현
- [x] 범용적인 **프런트엔드 품질 모듈**로 재사용 용이

---

## 2) 입력 데이터 (Input)
- **원본 입력:** UWB **TWR 거리** + **CIR 파형**
- **전처리(요약):** CIR 정규화/윈도우링 → 특징/시계열 입력 생성

**우리 20D 입력과의 매핑**

| 우리 20D 특징 | 논문 특징 | 변환/활용 |
|---|---|---|
| CIR/품질 | CIR 통계/시퀀스 | **DL→NLOS p** 산출 → **Rₜ 스케일** |
| Range | TWR d | 보정 전후 모두 유지 → 잔차 게이팅 |
| IMU/맵 | — | (옵션) 품질 융합 prior로 사용 |

- **호환성 평가:** ☑ **직접 사용 가능**

## 3) 모델 아키텍처 (Architecture)
- **파이프라인:** **[CIR] → [DL(NLOS p)] → [가중 KF(R/K 적응)] → [Position]**
- **핵심 모듈/수식:** \(p_{\text{NLOS}}\uparrow \Rightarrow R\uparrow,\ K\downarrow\)

**우리 모델과 비교**

| 구성 요소 | 논문 | 우리 모델 | 차이/시너지 |
|---|---|---|---|
| 품질추정 | DL 확률 | 퍼지/DPGMM/LSTM 등 | **앙상블**로 보정 신뢰↑ |
| 백엔드 | KF | KF/UKF/FG/PF | **R-스케일러 공통화** |

## 4) 학습/설정 (Training)
- **데이터/라벨:** CIR→(LOS/NLOS 또는 신뢰도)  
- **손실/옵티마이저/스케줄:** BCE/CE, Adam(예), plateau LR decay  
- **하이퍼파라미터:** 윈도우 길이, 특징 수, 임계 하한 \(R_{\min}\)

## 5) 실험 및 결과 (Experiments)
- **환경/셋업:** 실내 사용자 동작 다양(손/포켓/차폐)

| 지표 | 값 | 조건/비고 |
|---|---:|---|
