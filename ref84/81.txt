# 논문 #81: Distributed TDMA for Mobile UWB Network Localization (IEEE IoTJ, 2021)

## 1) 기본 정보
- **제목:** Distributed TDMA for Mobile UWB Network Localization
- **저자:** Yanjun Cao, Chao Chen, David St-Onge, Giovanni Beltrame
- **저널/학회:** IEEE Internet of Things Journal 8(17):13449–13459
- **출판연도:** 2021
- **DOI/링크:** 10.1109/JIOT.2021.3066243
- **인용 횟수:** —

**3줄 요약**
1. **완전 분산 TDMA**로 **멀티홉/모바일 UWB MANET**에서 **충돌 없는 고이용률** 스케줄링 달성.  
2. **2-hop 정보** 활용 **두 개의 슬랏 사이클(A/B)** 프레임과 **빠른 슬롯 할당 알고리즘** 제안.  
3. **동기화·TDMA·상대위치화·글로벌 맵 병합**까지 **UWB만으로** 구현, **지하 주차장 실기기** 데모.  

**핵심 기여(Contribution)**
- [x] **고속 수렴**·**풀 채널 사용** 지향 **분산 TDMA**  
- [x] **숨은 단말(2-hop)** 충돌 회피 스케줄  
- [x] **상대 위치화 + 맵 병합** 통합 파이프라인

---

## 2) 입력 데이터 (Input)
- **원본 입력:** **TWR** 거리, UWB 브로드캐스트 패킷(슬롯 상태/범위결과)
- **전처리 (pseudo)**  
  1) 프레임 시작 **분산 동기화**(최소 ID 기준 오프셋)  
  2) **Own/Extra/Passive** 슬롯 구분, **A/B 두 사이클** 운용  
  3) **cS(후보슬롯)/sS(송신슬롯)** 교환 → **충돌없는 Extra 슬롯 배분**

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| 다중 태그 운영 | 분산 TDMA 스케줄링 | RT 업데이트율 상향 |
| 품질/가중 | 2-hop 충돌 회피 | 신뢰도 채널에 반영 |
| 지도/병합 | 로컬→글로벌 병합 | 멀티로봇/크라우드 확장

- **호환성 평가:** [x] 직접 사용 가능

[**동기화**] → [**분산 TDMA**] → [**Ranging & 브로드캐스트**] → [**로컬 맵**] → [**글로벌 맵 병합**]

**특수 모듈:** **고정 프레임 n슬롯(노드수)**, **2-hop 정보 전파(A/B사이클)**, **슬롯분배 알고리즘(형제집합·공유슬롯)**
