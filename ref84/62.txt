# 논문 #62: UWB LOS/NLOS identification in multiple indoor environments using deep learning methods (Physical Communication, 2022)

## 1) 기본 정보
- **제목:** UWB LOS/NLOS identification in multiple indoor environments using deep learning methods
- **저자:** Qingzhi Liu, Zhendong Yin, Yanlong Zhao, Zhilu Wu, Mingyang Wu
- **저널/학회:** Physical Communication 52
- **출판연도:** 2022
- **DOI/링크:** 10.1016/j.phycom.2022.101695
- **인용 횟수:** —

**3줄 요약**
1. **ININN**: **CNN‖GRU 병렬** 특징 추출 + **SE(squeeze-and-excitation)** 채널 가중으로 **CIR 기반** LOS/NLOS 식별.  
2. **환경 인식 포함(거주/오피스/산업 CM1–CM8)** → **다중 클래스**로 일반화 강화.  
3. **시뮬(IEEE 802.15.4a)** + **오픈 실측 데이터** 모두에서 **기존법 대비 우수** 성능.

**핵심 기여(Contribution)**
- [x] **CNN‖GRU 병렬 + SE**로 시공간 특징 동시 학습
- [x] **환경(시나리오) 인식** 추가로 전이성 향상
- [x] **Eb/No 변화**와 채널모델별 범위오차 거동 분석

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - 데이터 타입: **UWB CIR**
  - 차원: LOS/NLOS + 환경 다중분류
  - 시나리오: Residential/Office/Industrial(IEEE 802.15.4a CM1–CM8)
- **전처리 (pseudo)**
  1. CIR 다운샘플/정규화  
  2. **CNN**(공간) ‖ **GRU**(시간) → **SE** 채널 가중  
  3. 결합 임베딩 → 분류 헤드

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| CIR 통계/파형 | CNN‖GRU+SE | 동일 파이프라인 |
| 품질/게이팅 | LOS/NLOS+환경 확률 | **R/Q 가중**으로 매핑 |
| 범위 보정 | — | NLOS 가중·보정모듈에 연계 |
