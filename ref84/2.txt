- **저자:** Jinmin Yang, Chunhua Zhu
- **저널/학회:** Sensors 23:1455
- **출판연도:** 2023-01-28
- **DOI/링크:** 10.3390/s23031455
- **인용 횟수:** —

**3줄 요약**
1. **ADS-TWR(TOF)** 기반 2D 위치에서 **가중 중심(초기치)** → **Residual Weighting** → **Newton 단순화 반복**으로 해를 구함.  
2. **하이퍼볼릭 특성 문제(TDOA)**를 회피하고 **Hessian 비가역성** 이슈 없이 **z=0 스칼라화**로 좌표 업데이트.  
3. 코너 앵커(0,0)(100,0)(0,100)(100,100) 시뮬레이션에서 **노이즈 환경** 전반에 성능·수렴성 개선.

**핵심 기여(Contribution)**
- [x] **Residual Weighting**으로 초기치 보정(이상치 억제)
- [x] **다변량→두 단일변수 Newton** 변환(미분 단순)
- [x] **임계/조합 수 \(N\)** 최적화 가이드(예: **역잔차 > 0.0031** 유지)

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - 데이터 타입: **ADS-TWR** 거리 \(R_n\)
  - 앵커 수: 4(2D)
- **전처리 방법 (pseudo)**
  1. 4원 배치의 **가중 중심**으로 \((x_{o1},y_{o1})\) 산출  
  2. **Residual Weighting**: \(R(x_{o1},y_{o1})^{-1}\) 가중 평균(역잔차 임계 **0.0031**)  
  3. **Newton** 반복: \(z(x,y)=0\)으로 변환, \(\Delta x,\Delta y\) 수렴 기준 적용
- **우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| Range | ADS-TWR \(R_n\) | 동일 |
| 품질(R/Q) | 역잔차 가중 | 가중치로 직접 활용 |
| 초기화 | 가중 중심 | 우리 초기화 모듈에 치환

- **호환성 평가:** [x] 직접 사용 가능
