# 논문 #56: UWB Indoor Localization Using Deep Learning LSTM Networks (Applied Sciences, 2020)

## 1) 기본 정보
- **제목:** UWB Indoor Localization Using Deep Learning LSTM Networks
- **저자:** Alwin Poulose, Dong Seog Han
- **저널/학회:** Applied Sciences
- **출판연도:** 2020
- **DOI/링크:** 10.3390/app10186290
- **인용 횟수:** —

**3줄 요약**
1. **TOA 기반 거리 3개(앵커 3)** 시퀀스를 입력으로 **2-층 LSTM**이 **사용자 위치(x,y)**를 직접 회귀.  
2. **IEEE 802.15.4a 채널**·가우시안 펄스 시뮬로 **TOA→거리→LSTM** 파이프라인 구성, **하이퍼파라미터 탐색**(lr/optimizer/loss/batch/hidden/timesteps).  
3. 전통 기법(LLSE/FPE/ML/MLE/WCE 등) 대비 **평균 위치오차 ≈ 7 cm** 달성(시뮬).

**핵심 기여(Contribution)**
- [x] 거리→좌표 **end-to-end LSTM** 회귀
- [x] **하이퍼파라미터 영향** 체계적 분석
- [x] 소수 앵커(3개) 시뮬에서 **정확도 개선**

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - 데이터 타입: UWB **TOA/거리**(앵커 3)
  - 차원: 2D 좌표
  - 샘플링: 예) 20 GHz, 펄스폭 0.5 ns(시뮬 설정)
- **전처리 방법 (pseudo)**
  1. **수신 신호 평균화 → 상관 → 피크 검출**로 TOA  
  2. **TOA→거리** 변환  
  3. **[d1,d2,d3]\_t…** 시퀀스 → 정규화

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문의 특징 | 변환 방법 |
|---|---|---|
| Range(거리) | 3앵커 거리 시퀀스 | 다앵커/20D 중 Range 서브셋 추출 |
| 품질/상태 | — | LSTM 입력 전 품질 필터 가능 |
| NLOS | (시뮬 채널 반영) | NLOS 확률 가중과 결합 |
