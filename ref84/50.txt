# 논문 #50: Real-time Indoor Localization and Safety Applications Using UWB in Construction (KSCE J. Civ. Eng., 2025)

## 1) 기본 정보
- **제목:** Real-time indoor localization and safety applications using UWB in construction
- **저자:** Jungmin Cho, Seungjin Jeong, Ju-Yong Kim, Gwang-Hee Kim, Jisun Lee, Byoungkil Lee
- **저널/학회:** KSCE Journal of Civil Engineering
- **출판연도:** 2025
- **DOI/링크:** 10.1016/j.kscej.2025.100164
- **인용 횟수:** —

**3줄 요약**
1. 실험실 **고정/운동(kinematic) 위치결정**과 **현장 적용**으로 UWB RTLS **정확도·견고성** 평가.  
2. 실험실 4앵커 고정에서 **2D RMSE ≤0.3 m**, **인체 차폐 영향 미미**; **면적 60→91.4 m²** 확대 시 **0.38→0.70 m(×1.85)**.  
3. **핸드오버** 상황에서 4앵커 **취약** → 현장 3앵커 **운동 RMSE ≤0.5 m**, **최대오차 1.5 m**; **삼각형 최장변 법선 방향** 오차 두드러짐.

**핵심 기여(Contribution)**
- [x] **핸드오버 시나리오**에서의 UWB 성능 분석
- [x] **앵커 수/배치**와 오차의 관계 규명(실험실→현장)
- [x] **EKF+삼변측량**으로 kinematic 평가
