# 논문 #41: Device-Free Indoor Localization of CSI Based on Limited Penetrable Horizontal Visibility Graph (IEEE Access, 2022)

## 1) 기본 정보
- **제목:** Device-Free Indoor Localization of CSI Based on Limited Penetrable Horizontal Visibility Graph
- **저자:** Ying Liu, Guoqing Li
- **저널/학회:** IEEE Access
- **출판연도:** 2022
- **DOI/링크:** 10.1109/ACCESS.2022.3187985
- **인용 횟수:** —
**3줄 요약**
1. Wi-Fi **CSI 서브캐리어 간 상관**을 **LPHVG**(Limited Penetrable Horizontal Visibility Graph)로 그래프화해 **디바이스 프리** 실내 위치추정.  
2. **네트워크(그래프) 특징 + 통계 특징**을 결합해 지문을 만들고 **SVR**로 회귀.  
3. **단일 AP**, **세 가지 실내 시나리오**에서 기존 기법 대비 정확도 유의 개선 및 **노이즈 강건성** 확인.

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - 데이터 타입: Wi-Fi CSI(30 subcarriers, amplitude/phase; 5300 NIC)
  - 차원: 2D 좌표 회귀
  - 샘플링: — (실험 세팅 내)
- **전처리 방법 (pseudo)**
  1. **위상 보정**(CFO/SFO 선형 보정)  
  2. **Hampel** 등으로 이상치 억제  
  3. **LPHVG**로 서브캐리어 그래프 구성 → **네트워크 특징** 추출  
  4. 통계 특징과 결합 → **SVR 회귀**

- **우리 20D 입력과의 매핑**

| 우리 특징 | 논문의 특징 | 변환 방법 |
|---|---|---|
| Fingerprint/Map | CSI 지문(LPHVG+통계) | 지문 모듈로 병렬 운용 |
| Range/NLOS | (비UWB) | 하이브리드 백업 채널 |
| 품질지표 | 네트워크 지표(평균 차수 등) | **품질스코어**로 R/Q 스케일러 연동 |

## 3) 모델 아키텍처 (Architecture)
- **전체 구조**  
[CSI 수집] → [위상보정/필터링] → [**LPHVG 그래프화**] → [네트워크+통계 특징] → [**SVR** 회귀] → [좌표]
- **특수 모듈**
  - 그래프: **LPHVG**(LPVG 대비 복잡도↓, HVG 대비 상관 보전↑)
  - Normalization: 표준화
- **파라미터 수**: 전통 ML(SVR) — 학습가능 파라미터 적음

- **우리 모델과 비교**

| 구성 요소 | 논문 | 우리 모델 | 차이점 |
|---|---|---|---|
| 센서 | **Wi-Fi CSI(단일 AP)** | UWB(범위/CIR) | 이기종 보조 채널 |
| 특징화 | LPHVG 네트워크 | CIR/통계/DL | 지문 레이어로 병렬 |
| 백엔드 | SVR 회귀 | EKF/UKF/PF/FG | late-fusion로 통합 |
