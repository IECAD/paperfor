# 논문 #72: Ultra Wideband (UWB) Localization Using Active CIR-Based Fingerprinting (IEEE Comm. Letters, 2023)

## 1) 기본 정보
- **제목:** Ultra Wideband (UWB) Localization Using Active CIR-Based Fingerprinting
- **저자:** Jaron Fontaine, Ben Van Herbruggen, Adnan Shahid, Sebastian Kram, Maximilian Stahlke, Eli De Poorter
- **저널/학회:** IEEE Communications Letters 27(5)
- **출판연도:** 2023
- **DOI/링크:** 10.1109/LCOMM.2023.3254146
- **인용 횟수:** —

**3줄 요약**
1. **한 번의 패킷**으로 각 앵커의 **CIR**만 수집하는 **능동 지문(fingerprinting)** 접근—**TWR 다패킷/동기 부담 제거**.  
2. **CNN + 앵커 서브셋(R=3, K=7)** + **Savitzky–Golay**로 경로 매끈화 및 **NLOS로 일부 CIR 누락** 상황도 위치 추정.  
3. **MAE 20.9–87.0 cm**, **P95 개선**, **TWR 대비 NLOS에서 우수**; **CIR 보간**으로 데이터 수집량 줄이면서 정확도 유지.

**핵심 기여(Contribution)**
- [x] **앵커-서브셋 학습**으로 **누락 CIR 견딤**  
- [x] **Savitzky–Golay** 후처리로 오차분위 개선  
- [x] **CIR 보간**으로 **라벨 수집 비용↓**

---

## 2) 입력 데이터 (Input)
- **원본 입력:** **CIR(1016 샘플, ~1 ns)**, 7 앵커(DW1000, fc≈4 GHz, BW 499.2 MHz), 태그 10 Hz
- **전처리 (pseudo)**
  1. **서브셋 생성**(R=3, K=7)  
  2. **CNN 회귀(x,y)** 학습(500 epoch, Adam 1e-3, bs=256)  
  3. **Savitzky–Golay**로 궤적 평활  
  4. **CIR 보간**으로 샘플 감축(SR=2,3) 보정

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| CIR 원시/통계 | CNN 회귀 입력 | 동일 파이프라인 연결 |
| 품질/게이팅 | 서브셋 다수결/불확실성 | R/Q 가중·앙상블화 |
| 라벨 비용 | 보간 | 데이터 수집 프로토콜에 반영

- **호환성 평가:** [x] 직접 사용 가능

## 5) 실험 및 결과 (Experiments)
| 설정 | 성과 |
|---|---|
| 동일 환경 | **MAE 20.9 cm**(최선) |
| 수정 환경 | **MAE 87.0 cm** |
