# 논문 #32: *Real-time indoor localization and safety applications using UWB in construction* (KSCE Journal of Civil Engineering, 2025)

## 1) 기본 정보
- **제목:** Real-time indoor localization and safety applications using UWB in construction
- **저자/소속:** —
- **저널/학회/연도:** KSCE Journal of Civil Engineering, 2025
- **링크/DOI:** —
- **키워드:** UWB RTLS, **건설현장**, EKF+삼변측량, **핸드오버 시나리오**, 앵커 구성

**3줄 요약**
1) 랩(4 앵커) **정적/운동** 성능과 **현장(3 앵커) 실시간 운동** 성능 분석.  
2) **랩:** 2D RMSE **≤ 0.3 m**(그림자 인체 영향 미미), **면적 60→91.4 m²** 시 **RMSE 0.38→0.70 m**(≈**1.85×**).  
3) **현장 3 앵커:** **2D RMSE ≤ 0.5 m**, 최대오차 **1.5 m**; **핸드오버/기하 취약 방향**에서 오차 집중 → **배치 최적화** 필요.

**핵심 기여(Contribution)**
- [x] **핸드오버**(LOS↔NLOS 급전이)에서의 RTLS 한계 정량화  
- [x] **앵커 수/배치/면적**이 정확도에 미치는 영향 분석  
- [x] 건설안전 적용 사례와 **실시간 EKF 파이프라인**
