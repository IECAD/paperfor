# 논문 #57: Attention-based LSTM Framework for Robust UWB–INS Integration in NLOS (Scientific Reports, 2025)

## 1) 기본 정보
- **제목:** Attention based LSTM framework for robust UWB and INS integration in NLOS environments
- **저자:** Meilin Ren, Junyu Wei, Jiangyi Qin, Xiaojun Guo, Haowen Wang, Shiqi Li
- **저널/학회:** Scientific Reports
- **출판연도:** 2025
- **DOI/링크:** 10.1038/s41598-025-05501-3
- **인용 횟수:** —

**3줄 요약**
1. **Attention-LSTM**이 **UWB 열화(NLOS)** 시 **의사(pseudo) 관측**을 생성해 **KF 업데이트**를 지속(루스/타이트 결합 모두).  
2. **표준화 혁신(χ²)** 기반 NLOS 탐지로 **LOS=학습 모드**, **NLOS=예측 모드** 전환.  
3. 실험적으로 **NLOS에서 위치오차 유의 감소**, **동적 시나리오 강건화**.

**핵심 기여(Contribution)**
- [x] **주의 메커니즘+LSTM**으로 시계열 중요 구간 가중
- [x] **의사 관측**으로 **측정 공백/바이어스** 보완
- [x] **루스/타이트 결합** 모두 적용 가능한 프레임워크

---

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - 데이터 타입: **IMU(가속/자이로/姿勢/속도/위치)** + 과거 **UWB 측정/증분**
  - 차원: 2D/3D 추정(플랫폼 일반)
- **전처리 방법 (pseudo)**
  1. **혁신 통계 \(d_k = v_k^\top S_k^{-1} v_k\)** → **NLOS 판정**  
  2. LOS: **교사신호**로 네트워크 갱신 / NLOS: **의사 관측** 생성  
  3. **KF(루스: 위치 증분 / 타이트: 범위 증분)** 업데이트

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| INS 보조 | Attention-LSTM 예측 | **pseudo-range/pos**를 20D 파이프라인에 주입 |
| 품질/게이팅 | 혁신 기반 NLOS | **R/Q 동적 조정** 규칙에 반영 |
| 결합 구조 | 루스/타이트 | 시나리오별 스위치 가능

- **호환성 평가**
  - [x] 직접 사용 가능

## 3) 모델 아키텍처 (Architecture)
[IMU+과거 UWB] → [**Attention-LSTM**] → [**의사 관측**] → [**Kalman Filter**] → [상태]

- **특수 모듈**
  - **Self-Attention(Q,K,V)**, 표준화 혁신 기반 **NLOS detector**

**우리 모델과 비교**

| 구성 요소 | 논문 | 우리 모델 | 차이점 |
|---|---|---|---|
| 학습 헤드 | Attention-LSTM | CNN/TCN/LSTM 혼합 | 헤드 대체/추가 |
| 게이팅 | 혁신-χ² | 규칙/ML 혼합 | 일원화 가능 |
