# 논문 #34: *A lightweight CIR-based CNN with MLP for NLOS/LOS identification in a UWB positioning system* (IEEE Communications Letters, 2023)

## 1) 기본 정보
- **제목:** A lightweight CIR-based CNN with MLP for NLOS/LOS identification in a UWB positioning system
- **저자/소속:** —
- **저널/학회/연도:** IEEE Communications Letters, 2023
- **링크/DOI:** —
- **키워드:** **CIR 원파형 CNN**, **수작업 특징+CNN 특징 융합(MLP)**, 경량, NLOS/LOS 분류

**3줄 요약**
1) **이미지 변환 없이 원시 CIR**로 **경량 CNN** 특징을 추출하고,  
2) **3개 신규 수작업 특징** 포함 **6개 수작업 특징**과 **MLP**로 융합 → **데이터 범람(길이 차)** 문제 해결,  
3) **계산량 7.39%**(vs 64×64 이미지 CNN), **정확도 +26.9%**(vs 기존 수작업), **+44.16%**(vs 이미지 CNN) 개선.

**핵심 기여(Contribution)**
- [x] **원시 CIR CNN**으로 경량·고효율 특징 추출  
- [x] **MLP 융합**으로 **수작업+CNN** 동시 활용  
- [x] 건물/오피스/**지하 광산** 3환경 검증

## 2) 입력 데이터 (Input)
- **원본 입력:** UWB **CIR(≈1k 길이)**  
- **전처리:** 1D Conv/Pool→**길이 81** 수준으로 압축 후 MLP 결합

**우리 20D 입력과의 매핑**

| 우리 20D 특징 | 논문 특징 | 변환/활용 |
|---|---|---|
| CIR/품질 | 원시 CIR CNN + 수작업 | **앙상블 품질 추정기**로 삽입 |
| NLOS score | 확률/로짓 | **Rₜ 스케일/게이팅** 연동 |
| Runtime | 경량 | 엣지 디바이스 배치 가능 |

- **호환성 평가:** ☑ **직접 사용 가능**

---

## 3) 모델 아키텍처 (Architecture)
- **파이프라인:** [CIR] → **1D CNN** 특징 → **MLP(수작업+CNN)** → **LOS/NLOS 확률**  
- **핵심 수치:** **연산량 7.39%**, 정확도 **+26.9% / +44.16%**

**우리 모델과 비교**

| 구성 요소 | 논문 | 우리 | 시너지 |
|---|---|---|---|
| 특징 | 1D CNN + 수작업 | CNN/TCN/통계 | **스태킹**으로 안정화 |
| 분류 | MLP | MLP/Softmax | 공통화 용이 |
| 배치 | 경량 | — | **모바일/엣지** 적합 |
