# 논문 #70: Deep Learning-Based Localization for UWB Systems (Electronics, 2020)

## 1) 기본 정보
- **제목:** Deep Learning-Based Localization for UWB Systems
- **저자:** Doan Tan Anh Nguyen, Han-Gyeol Lee, Eui-Rim Jeong, Han Lim Lee, Jingon Joung
- **저널/학회:** Electronics 9(10):1712
- **출판연도:** 2020
- **DOI/링크:** 10.3390/electronics9101712
- **인용 횟수:** —

**3줄 요약**
1. **CNN-LE**: **거리추정→좌표추정**을 분리하지 않고, **3개 수신기 신호로 만든 3채널 2D 이미지**를 입력해 **직접 (x,y) 회귀**.  
2. **IEEE 802.15.4a CM1–CM9** 채널 모델 및 **비대칭 영역(γ=D₁/D₂)**에서도 강건—**ToA+LS** 및 기존 CNN 대비 **RMSE 개선**.  
3. **하나의 CNN**으로 통합(기존 방법 대비 모델 수↓·오차 전파↓).

**핵심 기여(Contribution)**

- [x] **RGB 3채널 입력 이미지 생성법**(세 Rx 신호 오버샘플 2D화)  
- [x] 영역 형상 변화(복도형 등)에도 **성능 강건**

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - Rx=3개, **수신 UWB 신호**(매칭필터 후)  
  - 각 신호를 오버샘플링하여 **2D 이미지** 3개 생성 → **RGB 3채널** 스택
- **전처리 (pseudo)**
  1. \(y_i(t)=s(T_p−t)∗r_i(t)\) 샘플링  
  2. 정규화 후 **N×M** 이미지 리쉐이프  
  3. **[R,G,B]**로 결합

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| 멀티수신/CIR | 3채널 이미지 | 20D 중 신호표상 채널화 |
| 품질/상태 | — | 손실 가중/게이팅 병행 가능 |
| 좌표 추정 | CNN 회귀 | EKF/UKF와 하이브리드 가능

- **호환성 평가:** [x] 직접 사용 가능
