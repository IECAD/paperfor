# 논문 #77: Deep Learning-Based Positioning With Multi-Task Learning and Uncertainty-Based Fusion (IEEE TMLCN, 2024)

## 1) 기본 정보
- **제목:** Deep Learning-Based Positioning With Multi-Task Learning and Uncertainty-Based Fusion
- **저자:** Anastassios Foliadis, Mario H. Castañeda Garcia, Richard A. Stirling-Gallacher, Reiner S. Thomä
- **저널/학회:** IEEE Transactions on Machine Learning in Communications and Networking (TMLCN)
- **출판연도:** 2024-08-09 (current ver. 2024-08-26)
- **DOI/링크:** 10.1109/TMLCN.2024.3441521
- **인용 횟수:** —

**3줄 요약**
1. **CSI fingerprint** 기반 위치추정에서 **Early vs Late fusion** 비교—**Late + 불확실성 고려**가 역동적 환경에서 유리.  
2. **MTL(멀티태스크 학습)**으로 BS별 모델을 **공유/공통부**와 **개별부**로 나눠 공동 학습 → 성능·일반화 동시 개선.  
3. **Aleatoric/MCD(에피스테믹)** 불확실성 추정 + **스푸리어스 센서 억제** 융합으로 **최종 위치 결합** 안정화.

**핵심 기여(Contribution)**
- [x] **MTL + 불확실성 기반 Late fusion** 정식화  
- [x] **NLL(aleatoric)** + **MCD(epistemic)**로 좌표·불확실성 동시 추정  
- [x] **AUSE/IR**로 불확실성 신뢰성 평가

---

## 2) 입력 데이터 (Input)
- **원본 입력:** **CSI 행렬** \(H_n \in \mathbb{R}^{N_R \times N_C \times 2}\) (실·허수 스택, 정규화), **다중 BS**
- **전처리 (pseudo)**
  1. BS별 \(H_n\) 생성(실·허수 분리→스택→정규화)  
  2. **공유 백본(공통부)** + **BS별 헤드(개별부)**  
  3. **좌표 + \(\sigma_\alpha\)** 동시 회귀(NLL) + **MCD** 다중 추론

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 |
|---|---|---|
| 다중 센서/앵커 | **Late fusion** | 우리 앵커별 모델 → 가중 결합 |
| 품질/가중 | **불확실성(\(\sigma\))** | R/Q 가중치로 직접 사용 |
| 전이/일반화 | **MTL 공유부** | 공통 전처리 백본 공유

- **호환성 평가:** [x] 직접 사용 가능

---

## 3) 모델 아키텍처 (Architecture)
[CSI \(H_n\)] → [**공유 CNN/백본**] → [**BS별 헤드**] → [\(\hat{x},\hat{y}, \hat{\sigma}_x,\hat{\sigma}_y\)] → **Late Fusion(가중 평균 + 스푸리어스 억제)**

**특수 모듈:** **NLL**(aleatoric), **MCD**(epistemic), **스푸리어스 억제 융합**(타 센서와 불일치 시 가중 하향)

---

## 4) 학습 방법 (Training)
- **손실:** 좌표 MSE → **NLL 변형**으로 불확실성 학습  
- **전략:** **MTL**(BS=태스크) 가중 학습 + **Late fusion** 평가

---

## 5) 실험 및 결과 (Experiments)
- **결과 경향:** **Late+불확실성** > Early; **MTL**로 개별/최종 추정 모두 향상; **AUSE/IR**로 불확실성 신뢰성 검증
