# 논문 #76: Deep learning optimization positioning algorithm based on UWB/IMU fusion in complex indoor environments (Physical Communication, 2025)

## 1) 기본 정보
- **제목:** Deep learning optimization positioning algorithm based on UWB/IMU fusion in complex indoor environments
- **저자:** Zhou Zhaoxia, Xu Zhongwei*, Xia Jingbo
- **저널/학회:** Physical Communication 71:102702
- **출판연도:** 2025-05-12 (수리: 2025-04-30)
- **DOI/링크:** 10.1016/j.phycom.2025.102702
- **인용 횟수:** —

**3줄 요약**
1. **CNN–LSTM**으로 **UWB CIR**의 **LOS/NLOS 분류**를 선행.  
2. 분류 결과를 **DEKF(Distributed EKF)**와 융합해 **UWB+IMU** 통합 상태추정을 최적화.  
3. **오피스 환경**에서 **RMSE 0.205 m**, **MAE 0.192 m**, **NLOS에서도 안정성** 우수.

**핵심 기여(Contribution)**
- [x] **CNN–LSTM + DEKF** 결합 프레임
- [x] **분산 EKF**로 센서별 노이즈/동기 문제 완화
- [x] **UWB/IMU** 복합 환경 강건화

## 2) 입력 데이터 (Input)
- **원본 입력:** UWB **CIR**(분류용), IMU(가속/각속), UWB 거리/시간
- **전처리 (pseudo)**
  1. **CNN–LSTM**으로 LOS/NLOS 추정  
  2. **DEKF**: 로컬 예측→분산 업데이트로 UWB/IMU 융합  
  3. 궤적 산출(동기/드리프트 억제)

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 방법 |
|---|---|---|
| CIR/품질 | CNN–LSTM 확률 | R/Q 가중·게이팅 |
| 상태공정 | DEKF | EKF/UKF 대비 분산업데이트 도입 |
| 멀티센서 | UWB+IMU | 동일 파이프라인 확장
