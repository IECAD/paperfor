# 논문 #60: Domain-Adversarial Learning for UWB NLOS Identification in Dynamic Obstacle Environments (IEEE Sensors J., 2025)

## 1) 기본 정보
- **제목:** Domain-Adversarial Learning for UWB NLOS Identification in Dynamic Obstacle Environments
- **저자:** Qiu Wang, Ming-Song Chen, Xin Yan, Yong-Cheng Lin, Kai Li, Jia-Jie Liu, Chi-Zhou Zhang
- **저널/학회:** IEEE Sensors Journal 25(13)
- **출판연도:** 2025 (DOI 등록 2024)
- **DOI/링크:** 10.1109/JSEN.2024.3491178
- **인용 횟수:** —

**3줄 요약**
1. **EDANN-SOR**: **도메인 적대학습(DANN)** + **수기 CIR 특징(kurtosis, τ_med, τ_rms, FPSS 등)** 융합으로 **환경 간 분포차**를 정렬.  
2. **소량 타깃 데이터**만으로도 **이식/일반화**가 가능하도록 설계(도메인 판별기+GRL).  
3. **이종 환경**에서 **이진 LOS/NLOS ≥97.36%**, **다중클래스 97.07%** 달성.

**핵심 기여(Contribution)**
- [x] **DANN 기반** UWB NLOS **멀티클래스** 식별로 확장
- [x] **수기+심층** 혼합 특징으로 **도메인 불변** 표현 학습
- [x] **데이터 수집 비용↓**(타깃 소량 표본)

---

## 2) 입력 데이터 (Input)
- **원본 입력:** UWB **CIR**(LOS/여러 NLOS: 인체/유리/금속/벽 등)
- **특징화:** 원시 CIR + **통계/지연확산** 파생특징(τ_med, τ_rms, FPSS, RFPSD, TFRE …)

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 |
|---|---|---|
| CIR/통계 | 원시+수기 혼합 | 기존 20D에 파생 통계 채널 추가 |
| 품질/게이팅 | DANN 확률 | R/Q 동적 가중, 도메인 불변화 |
| 일반화 | 타깃 소량 적응 | 현장 교정 비용 절감

- **호환성 평가:** [x] 직접 사용 가능

---

## 3) 모델 아키텍처 (Architecture)
[CIR(+수기특징)] → [**Feature Extractor**] → { **Label Predictor**(LOS/NLOS/다중), **Domain Discriminator** }  
↳ **GRL(Gradient Reversal Layer)**로 **도메인 불변** 특성 학습

**우리 모델과 비교**

| 구성 요소 | 논문 | 우리 모델 | 차이점 |
|---|---|---|---|
| 전이/일반화 | DANN | 기본 습득/파인튜닝 | **환경 전이**에 특화 |
| 특징 | 원시+수기 | 원시 중심 | 혼합특징 병행 제안 |

---

## 4) 학습 방법 (Training)
- **손실:** CE(라벨) − λ·CE(도메인)  
- **데이터:** 소스 풍부, 타깃 **소량**(few-shot)  
- **전략:** 적대적 공동학습(λ 스케줄)

---

## 5) 실험 및 결과 (Experiments)
| 과제 | 성과 |
|---|---|
| LOS/NLOS 이진 | **≥97.36%** across scenarios |
| 다중클래스 | **97.07%** |
| 일반화 | 타깃 소량 표본으로 **성능 유지** |
