# 논문 #61: Application of an LSTM Algorithm Fused with Kalman Filter in UWB Indoor Positioning (Scientific Reports, 2024)

## 1) 기본 정보
- **제목:** Application of a long short-term memory neural network algorithm fused with Kalman filter in UWB indoor positioning
- **저자:** Yalin Tian, Zengzeng Lian, Penghui Wang, Mengqi Wang, Zhe Yue, Huabin Chai
- **저널/학회:** Scientific Reports 14:1925
- **출판연도:** 2024
- **DOI/링크:** 10.1038/s41598-024-52464-y
- **인용 횟수:** —

**3줄 요약**
1. **KF–LSTM**: **Kalman 필터**로 UWB 시계열 노이즈를 약화 → **LSTM**이 시계열 특성 학습해 **좌표 회귀**.  
2. **시뮬+실측**에서 **BP, KF-BP, LSTM** 대비 평균 위치정확도 **+71.31%, +37.28%, +49.31%** 향상.  
3. **노이즈가 클수록** 안정성 우위가 더 커짐.

**핵심 기여(Contribution)**
- [x] **KF 전처리 + LSTM** 결합으로 **잡음/불확실성** 완화
- [x] **DS-TWR** 원리·오차 식 기반 분석 포함
- [x] 시뮬과 **실측 데이터** 모두로 검증

---

## 2) 입력 데이터 (Input)
- **원본 입력:** UWB **DS-TWR** 거리 시퀀스(다중 앵커)
- **전처리 (pseudo)**
  1. **KF 예측/갱신**으로 거리시퀀스 필터링  
  2. 정규화된 윈도우 \([d_{1..N}]_{t−L..t}\) 구성  
  3. **LSTM** → \((x,y)\) 회귀

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문 특징 | 변환 |
|---|---|---|
| Range 시계열 | KF 필터링 | 동일, 사전 필터 모듈화 |
| 품질/게이팅 | — | NLOS 확률과 병용 가능 |
| 상태추정 | LSTM 회귀 | EKF/UKF/FG와 하이브리드

- **호환성 평가:** [x] 직접 사용 가능

---

## 3) 모델 아키텍처 (Architecture)
[**UWB Range (KF)**] → [**LSTM**] → [FC] → [\(\hat{x},\hat{y}\)]

**우리 모델과 비교**

| 구성 요소 | 논문 | 우리 모델 | 차이점 |
|---|---|---|---|
| 전처리 | KF | 가우시안/로버스트 혼합 | 파이프라인 정교화 |
| 백엔드 | LSTM 회귀 | EKF/UKF/NN 혼합 | 융합 가능(late/dual) |

## 4) 학습 방법 (Training)
- **손실:** MSE(좌표)
- **설정:** 시뮬 + 실측 혼합, 하이퍼 파라미터 표기

## 5) 실험 및 결과 (Experiments)
| 비교 | 평균 정확도 향상 |
|---|---:|
| vs BP | **+71.31%** |
| vs KF-BP | **+37.28%** |
| vs LSTM | **+49.31%** |

- **관찰:** 데이터 **노이즈↑ → KF–LSTM 안정성↑**
