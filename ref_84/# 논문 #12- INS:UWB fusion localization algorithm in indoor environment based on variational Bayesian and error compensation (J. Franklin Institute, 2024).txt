# 논문 #12: INS/UWB fusion localization algorithm in indoor environment based on variational Bayesian and error compensation (J. Franklin Institute, 2024)

## 1) 기본 정보
- **제목:** INS/UWB fusion localization algorithm in indoor environment based on variational Bayesian and error compensation
- **저자:** Long Cheng, Ke Liu
- **저널/학회:** Journal of the Franklin Institute 361:107204
- **출판연도:** 2024-08-28(online)
- **DOI/링크:** 10.1016/j.jfranklin.2024.107204
- **인용 횟수:** —

**3줄 요약**
1. **VBGMM**으로 **UWB 거리**를 **LOS/NLOS 군집** 분리, **SURE-Shrink**로 **INS 노이즈** 전처리.  
2. **MEF-GPDAF**(최대엔트로피 퍼지-일반화 확률 데이터 연관 필터)로 초기 융합 → **가상 관성 포인트** 기반 **오차 보상** → **UKF** 최종 정제.  
3. 실제 실험에서 **평균 위치 정확도 61.12% 개선**.

**핵심 기여(Contribution)**
- [x] **VBGMM** NLOS 완화 + **SURE-Shrink** INS 전처리
- [x] **MEF-GPDAF → 보상 → UKF** 3단 융합
- [x] 실제 데이터 **>60% 개선** 보고

## 2) 입력 데이터 (Input)
- **원본 입력:** UWB ToF(LOS/NLOS 혼재), INS(가속/자이로)
- **전처리 (pseudo)**
  1. **VBGMM**으로 거리 샘플 군집(LOS/NLOS)  
  2. INS는 **SURE-Shrink** 임계로 노이즈 억제  
  3. **MEF-GPDAF** 융합 → **가상 관성 포인트** 오차보상 → **UKF** 정제

| 보정 | 좌표 보상 | 잔차 기반 오프셋 보정 채널
