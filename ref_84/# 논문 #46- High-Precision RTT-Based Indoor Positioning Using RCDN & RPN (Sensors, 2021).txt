# 논문 #46: High-Precision RTT-Based Indoor Positioning Using RCDN & RPN (Sensors, 2021)

## 1) 기본 정보
- **제목:** High-Precision RTT-Based Indoor Positioning System Using RCDN and RPN
- **저자:** Ju-Hyeon Seong, Soo-Hwan Lee, Won-Yeol Kim, Dong-Hoan Seo
- **저널/학회:** Sensors 21(11):3701
- **출판연도:** 2021
- **DOI/링크:** 10.3390/s21113701
- **인용 횟수:** —
**3줄 요약**
1. **Wi-Fi RTT(FTM)** 거리의 수신 불안정/다중경로를 **RCDN(CNN)**으로 **보정**,  
2. 보정 거리들을 **RPN(RNN 기반, 구역 제안)**으로 실시간 위치 추정(**fingerprint-like**)  
3. 스캔 윈도우(10샘플)로 **수신율↑**, end-to-end로 **정확도 향상**.

**핵심 기여(Contribution)**
- [x] **RTT 보정 네트워크(RCDN)** 설계(스캔 스텝)  
- [x] **RPN**으로 실시간 위치 추정(분할/시계열)  
- [x] **엔드투엔드** 지연/유실/잡음 통합 처리

---

## 2) 입력 데이터 (Input)
- **원본 입력 형태**
  - 데이터 타입: **Wi-Fi RTT(TOF/TWR)** 거리(≥3 AP)
  - 차원: 2D 좌표
  - 샘플링: 빠른 주기(예: 0.02 s)
- **전처리 방법 (pseudo)**
  1. AP별 **RTT 10샘플 스캔 → CNN 보정(RCDN)**  
  2. 보정 거리들 분할/정규화  
  3. **RPN(RNN)**으로 구역 제안 → 위치

**우리 20D 입력과의 매핑**

| 우리 특징 | 논문의 특징 | 변환 방법 |
|----------|-------------|----------|
| 비UWB 보조 | RTT 기반 | **보조 채널**로 late-fusion |
| 품질지표 | 보정 RMSE | **R/Q 스케일러** 보조 |
| 맵 | (불필요) | 맵 없는 보조 추정

- **호환성 평가**
